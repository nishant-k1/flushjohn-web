## What You CAN Do Now

### 1. Use Your Dover Address for GBP (Service-Area Business)
**You CAN create a Google Business Profile as a "Service-Area Business"**

- Use your Dover, DE address (8 The Green STE R, Dover, DE 19901)
- Mark it as "Service-Area Business" (not a physical location)
- Add all your service cities
- Google will mail a postcard to verify - use a mail forwarding service or virtual mailbox
- **Services:** iPostal1, Anytime Mailbox, EarthClassMail ($10-30/month)

**Steps:**
1. Go to business.google.com
2. Create profile with Dover address
3. Select "I deliver goods and services to my customers"
4. Add all service areas (25+ cities)
5. Request verification postcard
6. Use mail forwarding service to receive code
7. Verify and publish

### 3. Build Reviews Over Time

#### Strategy:
1. **After each successful rental:**
   - Send follow-up email 2-3 days after delivery
   - Ask for feedback
   - Include link to Google Business Profile review
   - Offer small incentive (5% off next rental) for review

2. **Make it easy:**
   - Create a review request page
   - Send SMS follow-ups
   - Include review links in confirmation emails

3. **Respond to all reviews:**
   - Thank positive reviewers
   - Address any concerns quickly
   - Shows you care
